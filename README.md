# Amazon-Sales-Analysis
Amazon Sales Analysis | Python (Pandas, NumPy, Matplotlib, Seaborn) & Power BI | Data Cleaning, EDA, Sales Trend Analysis, Interactive Dashboard Development

**Project Overview:**
This project analyzes Amazon sales data using Python and Power BI to uncover sales trends, product performance, and customer purchasing patterns. The objective was to transform raw sales data into meaningful insights that support data-driven business decisions.

**Tools Used**
Python (Pandas, NumPy, Matplotlib, Seaborn)
Power BI
Project Tasks
Cleaned and prepared raw sales data using Python.
Performed Exploratory Data Analysis (EDA) to identify trends and patterns.
Analyzed sales performance across products, categories, and regions.
Created visualizations and interactive dashboards in Power BI.
Generated business insights and recommendations based on the analysis.

**Key Insights**
Identified top-performing products and categories by sales.
Analyzed monthly sales trends and seasonal fluctuations.
Evaluated regional sales performance and customer demand patterns.
Highlighted opportunities for improving revenue and inventory planning.

**Project Outcome**
Developed an interactive Power BI dashboard and data-driven insights that help stakeholders monitor sales performance and make informed business decisions.
